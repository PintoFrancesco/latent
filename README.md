# Latent Series

For years I looked at medical images. CT scans, PET scans, ultrasounds, X-rays; images of a body that works, breaks down, adapts. What they show is rarely simple: the boundary between growth and pathology is not a rupture of rules but the same formula applied under different conditions.

This series comes from that fascination and from that fear.

Cells grow and divide according to mathematical logic. Neurons branch outward searching for connections in a space that does not yield. A virus inserts itself into the host's logic and bends it from within without rewriting its rules. The tumour cell uses the same logic but has lost the cooperative signal that imposes the limit. It reorganises around its own gradients: periphery replicating at maximum rate, intermediate zone accumulating survival mutations, necrotic core at the centre. A different order built only to persist.

Those logics become code. The output is a fine art print: every state of the form's evolution remains visible, accumulated in a single frame. Nothing disappears, the image carries within it the entire history of the form, every fold, every deviation, every trace of the path that generated it. From a distance it is an organism. In the detail, each zone opens another scale, another form.

*Morphogen*, *Dendrite*, and *Phage* are different configurations of the same question applied to distinct biological systems: the molecular signal that initiates growth, the neural structure searching for space, the external agent that transforms from within.
